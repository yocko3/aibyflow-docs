# ご利用ガイド

AibyFlowの概要と、ツールの購入・利用方法について説明します。

## AibyFlow 概要
AibyFlowは、AIツールの作成から販売までを最短距離でつなぐサービスです。

- **AIによるコード生成**: n8nワークフローのJSONを貼り付けると、AIがそれをプロンプトとして解釈し、実行可能なPythonコードを生成します。
- **フォーム自動作成**: 専門知識がなくても、ユーザー入力用のフォームが簡単に作成でき、サイト上で実行できます。
- **ツールの販売**: 作成したツールはそのまま販売ページとして公開し、収益化が可能です。
- **Solana決済**: 決済にはSolanaブロックチェーンを採用しています。

### ⚠️ プロトタイプ版の制限事項
現在はプロトタイプフェーズのため、以下の状態となっています。
- サイト側が用意したプリセットツールの購入・利用のみ可能です。
- ユーザー自身によるPythonコードの生成機能は現在準備中です。

## ツール購入・実行方法
アカウント作成は不要です。ウォレットを接続するだけで、プライバシーを守りながら利用できます。

### 1. ウォレットの接続
1. 商品ページにある **「Select Wallet」** ボタンをクリックします。
2. お使いのウォレットを選択して接続してください。
   - **おすすめ**: Phantomウォレット（Solanaを扱えるものであればMetaMask等でも可能です）。

### 2. フォーム入力と実行
1. ツールの入力フォームに必要な情報を入力します。
2. **「実行してプレビューを作成」** ボタンをクリックします。

### 3. お支払い
1. ウォレットが立ち上がるので、支払いを確認します。
   - **ガス代について**: 0 SOLの商品であっても、Solanaネットワークの利用料（非常に少額のガス代）は発生します。
2. 支払いが完了するとツールが実行されます。

### 4. 実行結果の確認
1. ツールの実行が終わると、結果が画面にプレビューされます。
2. 必要に応じて結果をダウンロードしてください。

> [!IMPORTANT]
> **返金について**: ツールの実行が失敗した場合は自動的に返金処理が行われ、ガス代もかかりません。安心してお試しください。
