# AibyFlow ドキュメント

**AibyFlowへようこそ！**

AibyFlowは、n8nのワークフローをベースに、誰でも一瞬でAIツールを構築・販売できるプラットフォームです。

## AibyFlowでできること
- **n8n連携**: ワークフローのJSONを貼るだけでPythonコードを自動生成。
- **簡単公開**: フォームを自動作成し、サイト上ですぐに実行可能。
- **収益化**: 作成したツールをそのまま販売。
- **Solana決済**: 暗号資産（SOL）によるスムーズな決済。

> [!TIP]
> 現在は**プロトタイプ版**として、運営が用意したツールの販売・実行機能を提供しています。
